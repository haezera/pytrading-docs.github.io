pytrading documentation


Welcome to the documentation of the pytrading API.

Pytrading uses a simple Flask HTTP server, and serves jsonified data to the HTTP layer.

To use Pytrading's API, simply download the package from github, and install all dependencies necessary. Afterwards, run the server.py file, and you should be good to go!

The API was designed with consideration of the usage of a React frontend - however, you may use it in whichever way you see fit.
